C语言快速复习——文件流

​ 我们现在开始作简单的文件操作的复习

fopen函数

​ 打开文件的函数，也就是说我们需要创建一个FILE*指针来接受文件

struct _iobuf {
        char *_ptr;
        int   _cnt;
        char *_base;
        int   _flag;
        int   _file;
        int   _charbuf;
        int   _bufsiz;
        char *_tmpfname;
        };
typedef struct _iobuf FILE;

​ 为了方便我们的接受文件，我们选择使用这个fopen函数.

FILE* fp = fopen(fileAddr,Reading mode)

​ 那么，这就是一个重要的问题：我们应该怎么样打开文件呢？

w open for writing (file need not exist) 如果文件不存在，就创建一个新的。
a open for appending (file need not exist)
r+ open for reading and writing, start at beginning
w+ open for reading and writing (overwrite file)
a+ open for reading and writing (append if file exists)
r open for reading

fclose函数

​ 对于文件指针，我们需要注意的是文件使用完毕之后，需要关闭文件指针。

fclose(FILE* filePointer);

​

fgetc fputc

​ 跟getc 和 putchar 是一回事情！向指定的文件读入或者输出一个字符

fseek

​ 我们在流中定位一个指针的位置：这只是适用在 fseek 里头！

SEEK_SET 0// AT THE BEGINNING OF THE FILE
SEEK_CUR 1// AT THE CURRENT PLACE OF THE FILE
SEEK_END 2// AT THE END OF THE FILE

fputs fgets fprintf fscanf

​ 还是我们说的，跟puts gets printf scanf一回事

feof()

​ 到达文件尾时返回真值

rewind()

​ 复位文件定位器到文件开始处

fread()

​ 从流中读指定个数的字符
